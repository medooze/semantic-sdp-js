▸ answer(supported)

Helper usefull for creating media info answers.

  • Will reverse the direction
  • For each supported codec, it will change the payload type to match the offer and append it to the answer
  • For each supported extension, it will append the ones present on the offer with the id offered
answer(supported: Object): MediaInfo
Parameters
supported (Object) Supported codecs and extensions to be included on answer
Name Description
supported.codecs Map<String, CodecInfo> List of strings with the supported codec names
supported.extensions Set<String> List of strings with the supported codec names
